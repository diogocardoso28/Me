Aria

The smart home with assistant "Aria" was a project developed as the final project for my professional course, it featured multiple arduino based sensors (wich comunicated over MQTT and WiFi). It used MySQL for database and NODE for the backend (saving values to database, handling real time events, registering new devices, etc). I also built and integration compatible with Google Assistant for easier control of the home. It also featured many 3d props desiged by me.

RFID Reader and Cloner

Project built using an ESP32 paired with an RC-522 module. This project features a nice looking ui with real time actions (such as when new card read). It can store data from multiple cards. Option such as wifi hotspot name, and target wifi network can be changed trough UI and made persistent using the built in file system (SPIFFS).

3D Printer LED Ring

I built this portable status ring to facilitate the monotoring of my 3D printer, it was built using a Wemos d1 mini and a 16 LED neopixel ring. It was codede using the esphome platform and integrates into home assistant to get the 3d printer data. The printer is connected to a server running octoprint. This ring displays job percentage and when warming up the percentage that it completed (target temperature vs ambient temperature).
